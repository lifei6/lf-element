Button 按钮 ​

常用的操作按钮。

基础用法 ​

使用 type、plain、 round和 circle来定义按钮的样式。

<template>
  <p>
    <lf-button>Default</lf-button>
    <lf-button type="primary">Primary</lf-button>
    <lf-button type="success">Success</lf-button>
    <lf-button type="info">Info</lf-button>
    <lf-button type="warning">Warning</lf-button>
    <lf-button type="danger">Danger</lf-button>
  </p>

  <p>
    <lf-button plain>Plain</lf-button>
    <lf-button type="primary" plain>Primary</lf-button>
    <lf-button type="success" plain>Success</lf-button>
    <lf-button type="info" plain>Info</lf-button>
    <lf-button type="warning" plain>Warning</lf-button>
    <lf-button type="danger" plain>Danger</lf-button>
  </p>

  <p>
    <lf-button round>Round</lf-button>
    <lf-button type="primary" round>Primary</lf-button>
    <lf-button type="success" round>Success</lf-button>
    <lf-button type="info" round>Info</lf-button>
    <lf-button type="warning" round>Warning</lf-button>
    <lf-button type="danger" round>Danger</lf-button>
  </p>

  <p>
    <lf-button icon="search" circle />
    <lf-button type="primary" icon="edit" circle />
    <lf-button type="success" icon="check" circle />
    <lf-button type="info" icon="message" circle />
    <lf-button type="warning" icon="star" circle />
    <lf-button type="danger" icon="trash" circle />
  </p>
</template>

图标按钮 ​

使用 icon属性来定义按钮的图标。

<template>
  <div>
    <lf-button type="primary" icon="edit" />
    <lf-button type="primary" icon="share" />
    <lf-button type="primary" icon="trash" />
    <lf-button type="primary" icon="search">Search</lf-button>
    <lf-button type="primary"> Upload<lf-icon icon="upload" style="margin-left: 8px" /> </lf-button>
  </div>
</template>

禁用状态 ​

使用disabled禁用按钮。

<template>
  <p>
    <lf-button>Default</lf-button>
    <lf-button type="primary">Primary</lf-button>
    <lf-button type="success">Success</lf-button>
    <lf-button type="info">Info</lf-button>
    <lf-button type="warning">Warning</lf-button>
    <lf-button type="danger">Danger</lf-button>
  </p>
  <p>
    <lf-button disabled>Default</lf-button>
    <lf-button type="primary" disabled>Primary</lf-button>
    <lf-button type="success" disabled>Success</lf-button>
    <lf-button type="info" disabled>Info</lf-button>
    <lf-button type="warning" disabled>Warning</lf-button>
    <lf-button type="danger" disabled>Danger</lf-button>
  </p>
</template>

加载中状态 ​

使用 loading属性来定义按钮的加载状态。

TIP

您可以使用 loading 插槽或 loadingIcon 属性自定义您的 loading 状态时显示的图标。

ps: loading 插槽优先级高于 loadingIcon 属性。

<template>
  <div>
    <lf-button type="primary" loading>Loading</lf-button>
    <lf-button type="primary" loading-icon="circle-notch" loading>Loading</lf-button>
    <lf-button type="primary" loading>
      <template #loading>
        <lf-icon style="margin-right: 6px" icon="star" spin />
      </template>
      Loading
    </lf-button>
  </div>
</template>

按钮组 ​

使用 <lf-button-group> 对多个按钮分组。

<template>
  <p>
    <lf-button-group disabled>
      <lf-button type="primary" icon="arrow-left">Previous Page</lf-button>
      <lf-button type="primary">
        Next Page<lf-icon icon="arrow-right" style="margin-left: 8px" />
      </lf-button>
    </lf-button-group>
  </p>
  <p>
    <lf-button-group type="warning" size="small">
      <lf-button type="primary" icon="edit" round />
      <lf-button type="success" icon="share" />
      <lf-button type="danger" icon="trash" round />
    </lf-button-group>
  </p>
</template>

按钮尺寸 ​

使用 size属性来定义按钮的尺寸。

<template>
  <p>
    <lf-button size="large">Large</lf-button>
    <lf-button>Default</lf-button>
    <lf-button size="small">Small</lf-button>
    <lf-button size="large" icon="search">Search</lf-button>
    <lf-button icon="search">Search</lf-button>
    <lf-button size="small" icon="search">Search</lf-button>
  </p>
  <p>
    <lf-button size="large" round>Large</lf-button>
    <lf-button round>Default</lf-button>
    <lf-button size="small" round>Small</lf-button>
    <lf-button size="large" icon="search" round>Search</lf-button>
    <lf-button icon="search" round>Search</lf-button>
    <lf-button size="small" icon="search" round>Search</lf-button>
  </p>
  <p>
    <lf-button size="large" icon="search" circle />
    <lf-button icon="search" circle />
    <lf-button size="small" icon="search" circle />
  </p>
</template>

Tag ​

可以自定义元素渲染的标签。例如：button、div、路由链接等。

div tag
a tag 

<template>
  <lf-button>button tag</lf-button>
  <lf-button tag="div" role="button" tabindex="0">div tag</lf-button>
  <lf-button type="primary" tag="a" href="javascript:;" target="_blank" rel="noopener noreferrer">
    a tag
  </lf-button>
</template>

节流模式 ​

可以通过 useThrottle 属性来定义按钮是否使用节流模式 默认为 true。

<script setup lang="ts">
// import { ErMessage } from 'eric-ui'
function handleBtnClick() {
  // ErMessage.info('btn click')
  console.info('btn click')
}
</script>

<template>
  <p>
    <lf-button @click="handleBtnClick" :throttle-duration="2000"> with throttle</lf-button>
    <lf-button :use-throttle="false" @click="handleBtnClick">without throttle</lf-button>
  </p>
</template>
